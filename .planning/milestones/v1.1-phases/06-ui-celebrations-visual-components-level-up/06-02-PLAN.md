Implementation:

1. **Bottom sheet container:**
   - `if (!isOpen) return null;` for initial render
   - Track `isLeaving` state for exit animation
   - Backdrop: `fixed inset-0 z-50 bg-black/80 backdrop-blur-sm` with onClick → handleClose

   - Add body scroll lock: `useEffect` that sets `document.body.style.overflow = 'hidden'` when open, restores on cleanup
   - Add a drag handle indicator at top: small gray bar centered (cosmetic, not functional)

2. **Header section:**
   - "XP BREAKDOWN" title in doom-gold, text-[10px], tracking-widest, centered
   - Close button: "✕" in top-right corner, text-gray-500

3. **Tab switcher:**
   - `activeTab` state: `'week' | 'alltime'`, default 'week'
   - Two buttons in flex row with `border-b-2 border-gray-800`
   - Active tab: `text-doom-gold border-b-2 border-doom-gold`
   - Inactive tab: `text-gray-500`
   - Labels: "THIS WEEK" and "ALL TIME"

4. **"This Week" tab content:**
   - Call `getWeeklyXPBreakdown(workoutCount, currentStreak, weekStatus)` to get breakdown
   - Display rows with label (left) and value (right):
     - "WORKOUTS" → `+${breakdown.baseXP} XP` in white
     - "STREAK BONUS" → `×${breakdown.streakMultiplier}` in doom-gold (only show if > 1.0, otherwise show "×1.0" in gray)
     - Divider line
     - "WEEK TOTAL" → `+${breakdown.totalXP} XP` in doom-green, font-bold
   - If weekStatus is not 'normal', show message: "No XP earned during {weekStatus} weeks" in gray italic
   - Use `text-[9px]` for labels, `text-[11px]` for values

5. **"All Time" tab content:**
   - Calculate workoutXP = totalXP - achievementXP (derived, not fetched separately)
   - Display rows:
     - "WORKOUT XP" → `${workoutXP.toLocaleString()} XP` (includes streak bonuses)
     - "ACHIEVEMENT XP" → `${achievementXP.toLocaleString()} XP`
     - Divider line
     - "TOTAL XP" → `${totalXP.toLocaleString()} XP` in doom-green, font-bold

6. **Rank progression section** (below both tabs, always visible):
   - Divider line
   - "RANK PROGRESSION" header in doom-gold, text-[9px]
   - Current rank: full name with tier color, tagline in gray
   - Next rank: full name with tier color, `${xpToNextRank.toLocaleString()} XP to go` in gray
   - If max rank (nextRank is null): "MAXIMUM RANK ACHIEVED" in doom-gold
