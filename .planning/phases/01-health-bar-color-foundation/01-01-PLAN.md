<objective>
Replace the current confusing traffic light color scheme with an intuitive DOOM health bar paradigm where green represents best performance (full health), yellow represents moderate performance (damaged), and red represents critical performance (low health).

Purpose: Users currently find the color scheme counterintuitive (gold for best performance doesn't match DOOM health mechanics). The health bar paradigm provides immediate visual understanding: green = healthy, yellow = damaged, red = critical, matching the DOOM universe mental model.

Output: Centralized color utility functions in weekUtils.ts and updated Dashboard visualization using the new health bar color scheme with dual visual encoding for sick/vacation weeks (border for status, background for performance).
</objective>
